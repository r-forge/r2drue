petHgsm(r2dRue)R Documentation

Hargraves-Samani Potential Evapotranspiration function

Description

Given a set of images of temperature and solar radiation, the potential evapotranspiration is calculated according to Hargraves-Samani equation. batchPet () provides a batch process to calculate the PET over several sets of images.

Usage

petHgsm(Tmin,Tmax,Tmed,Rad,month)
batchPetHgsm(date, Tmin, Tmax, Tmed, Rad, outFl, ...)

Arguments

date A vector with the year and the month of the first image dataset. Like c(1970,1) for january 1970
month The month to calc the PET (numeric 1 to 12)
Tmin In petHgsm a SpatialGridDataFrame with minimum air temperature (Celsius degree) In batchpetHgsm a character vector with the names of the files with minimum air temperature (Celsius degree)
Tmax In petHgsm a SpatialGridDataFrame with maximum air temperature (Celsius degree) In batchpetHgsm a character vector with the names of the files with maximum air temperature (Celsius degree)
Tmed In petHgsm a SpatialGridDataFrame with mean air temperature (Celsius degree) In batchpetHgsm a character vector with the names of the files with mean air temperature (Celsius degree)
Rad In petHgsm a SpatialGridDataFrame with extra-terrestial solar radiation In batchpetHgsm a character vector with the names of the files with extra-terrestial solar radiation
outFl A list of file names to save the results
... Any unmatched parameter will be passed to the writeGDAL routine.

Value

petHgsm returns a spatialGridDataFrame representing the potential evapotranspiration. batchEtp write to disc, the calculated images for each set of images provided.

Author(s)

Gabriel del Barrio, Marieta E. SanJuan & Alberto Ruiz

References

Estimating potential evapotranspiration. Hargreaves, G.H., Samani, Z.A. Journal of the Irrigation & Drainage Division - ASCE Volume 108, Issue IR3, 1982, Pages 225-230
